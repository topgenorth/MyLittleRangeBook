# Overview

This started out just experimenting with the Garmin FIT SDK and Avalonia. Suddenly this has evolved into a logbook of when I go shooting (something I track in a spreadsheet).

This is primarily .NET 10/C#, SQLite with the Dapper micro ORM, and mostly CLI based. Once I have enough functionality stubbed out on the CLI, I will get back on the GUI part (Avalonia). Might just keep this as a TUI; I'm more concerned about functionality and working on older hardware.
